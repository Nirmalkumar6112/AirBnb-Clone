# AirBnb Clone
Full Stack AirBnb Clone built with Next.js

### Tools and Frameworks Used:
- MongoDb, Prisma, React, Next.js, Tailwind CSS 
- Authentication based on NextAuth, Google & GitHub OAuth
- Cloudinary CDN
